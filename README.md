### Getting started ###

1. `git clone git://github.com/concord-consortium/NodeLinkDemo.git`
2. `cd NodeLinkDemo`
3. `git submodule update --init --recursive` 
4. `bundle install --binstubs`
6. `bin/sc-server -v`
